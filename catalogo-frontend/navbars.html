Guía de Navbars con Tailwind CSS

1. Navbar Transparente

bg-transparent absolute top-0 text-white hover:text-gray-300

2. Navbar Oscuro

bg-gray-900 sticky top-0 shadow-lg text-gray-300

3. Glassmorphism

bg-white/10 backdrop-blur-md border border-white/20 fixed top-0

4. Sticky Effects

sticky top-0 navbar-scroll border-b-2 transition-all

5. Mobile First

md:hidden hidden md:flex border-t space-y-1

Consejos

• Usa sticky para mantener visible

• fixed para overlay completo

• z-50 para estar encima

• backdrop-blur para efectos glass
